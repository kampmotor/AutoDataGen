Architecture

Core Abstractions

abstract class
AutoSimPipeline (ABC)
Main pipeline orchestrator. Subclasses implement load_env() and get_env_extra_info().
+ run() → PipelineOutput
+ decompose() → DecomposeResult
+ execute_skill_sequence()
AutoSimPipelineCfg
abstract class
Skill (ABC)
Base class for atomic robot skills. Each skill has plan + step lifecycle.
+ plan(world_state, goal) → bool
+ step(world_state) → SkillOutput
+ extract_goal_from_info() → SkillGoal
SkillCfg SkillExtraCfg
abstract class
Decomposer (ABC)
Decomposes high-level tasks into structured subtask & skill sequences. LLM-based with filesystem cache.
+ decompose(env_extra_info) → DecomposeResult
+ is_cache_hit(task_name) → bool
+ read/write_cache()
DecomposerCfg
base class
ActionAdapterBase
Maps skill outputs to environment action tensors. Plugable per-skill via registration.
+ apply(skill, output, env) → Tensor
+ register_apply_method(name, fn)
ActionAdapterCfg
registration system
Pipeline & Skill Registry
Plugin-based registration system for pipelines and skills.
register_pipeline(id, entry_point, cfg_entry_point) make_pipeline(id, **cfg_overrides) → AutoSimPipeline @register_skill("name", "desc", cfg_type) SkillRegistry.create(name, extra_cfg) → Skill

Skills

Atomic Robot Skills

skill
ReachSkill
Motion planning to a target pose. Supports batch-planning with multiple candidate poses (K×7).
cuRobo CuroboSkillExtraCfg
skill
RelativeReachSkill
Reach relative to an object's frame axis. Used for orientation-dependent manipulation.
cuRobo CuroboSkillExtraCfg
skill
GraspSkill
Gripper control skill. Opens/closes end-effector based on object interaction.
no planner
skill
NavigateSkill
Mobile base navigation: A* global path planning + DWA local obstacle avoidance.
A* / DWA NavigateSkillExtraCfg
skill
RetractSkill
Retract manipulator to a safe pose after task completion.
cuRobo CuroboSkillExtraCfg
skill
RotateSkill
Rotate end-effector in-place. Used for orientation adjustment without translation.
cuRobo CuroboSkillExtraCfg
Capability Path Purpose
CuroboPlanner capabilities/motion_planning/curobo/ GPU-accelerated IK & motion planning via cuRobo
AStarPlanner capabilities/navigation/a_star/ Global path planning on occupancy grid
DWAPlanner capabilities/navigation/dwa/ Local obstacle avoidance & trajectory rollout
OccupancyMap capabilities/navigation/occupancy_map.py Grid map builder with inflation buffer
